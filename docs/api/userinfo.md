# Userinfo

## Success

```json
{
    "status": "success",
    "result": {
        "username": "some_name"
    }
}
```

## Error

```json
{
    "status": "error",
    "result": "You must be logged in to make a request"
}
```

Errors:

- Unauthorized request: You must be logged in to make a request
- Server Error: We are sorry. There are some issues on our side. Please try again later
